Scraperr Configuration

Instructions for Scraperr Configuration:

  1. Configure your environment variables using the form above
  2. Click "Generate Secret Key" to create a new random secret key
  3. Click "Save Configuration" to generate your .env file
  4. Copy the generated .env file content to /home/ncx/Proyectos/scraperr-local/.env
  5. Restart your Docker containers using: make down && make up
